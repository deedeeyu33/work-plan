一创业总体定位

1. 长期公司定位

建立一家由本人独立拥有和管理、以中国供应链为基础、面向美国市场开展产品销售、贸易、市场进入及相关商业服务的公司。

公司长期不局限于：工业设备、激光清洗机、塑料加工行业、单一产品、单一销售模式。

未来可能开展：中国产品在美国的进口、代理和销售；针对美国企业的产品与解决方案；产品选型、试用、培训和售后；中国供应链采购与美国市场进入服务；品牌代理；其他通过市场验证的新产品与新项目。

2. 第一阶段业务切入口
第一阶段项目：以激光清洗设备为工具，为美国滚塑、塑料加工、橡胶加工及模具相关企业提供模具清洗、残留物去除和生产效率改善解决方案。

2026年优先客户：滚塑生产企业、其他塑料加工企业、橡胶加工企业、模具制造/维修和使用企业。暂不主动开发其他激光清洗行业。

3. 公司提供的不只是机器
  1. 了解客户当前如何清洗模具
  2. 判断污染物、材料和表面是否适合激光清洗
  3. 进行样品试洗或后续现场验证
  4. 比较传统方法和激光清洗的时间、人工、耗材及停机成本
  5. 推荐适合的设备型号和参数
  6. 销售设备和必要的安全配套
  7. 提供操作培训
  8. 提供有限保修、远程诊断和备件支持
  9. 根据同一批客户的真实需求，未来增加相关测试、维护或辅助设备

二2026年创业执行原则

1. 公司做宽，业务做窄

公司名称保持宽泛；2026年对外销售表达保持具体；不把公司永久限定为激光设备公司；也不向客户表达为"什么都卖的贸易公司"。

2. 必须完成的内容
  • 学会操作和基础调试机器
  • 完成真实工厂案例
  • 验证美国客户需求
  • 注册美国公司并建立收款能力
  • 个人账户和公司账户分离
  • 正式进口前确认产品合规和安全
  • 正式销售前明确合同、保修和售后责任
  • 算清售价、落地成本、毛利和现金流
  • 明确本人能够在美国开展哪些活动
3. 可以推迟的内容

2026年前期暂不处理：独立产品品牌、联邦商标注册、多个DBA、第二家LLC、豪华网站、办公室、长期仓库、大量库存、助理、全职员工、全职销售团队、其他行业探索、同时开发多条产品线。

4. 投入顺序
证明设备有效 → 做出可信案例 → 证明目标客户有问题 → 证明客户愿意试用或进一步了解 → 确认进口与安全可行 → 证明客户愿意付钱 → 证明毛利和售后可承担 → 扩大销售、产品和团队

三2026年至2029年长期目标

以下数字是第一版经营假设，必须在2026年10月17日完成美国市场验证后重新评估。

2026年：完成基础搭建与首轮验证

产品与客户：掌握设备操作，完成1个真实工厂案例，10~15次有效客户访谈，3~5个合格客户机会，2~3家愿意现场演示或生产试用。

公司：成立100%持有的美国公司，取得EIN，建立公司和个人银行账户，建立会计记录，明确进口销售路径，完成E-2可行性评估。

财务：不以盈利为标准，算清售价、单台落地成本、毛利、试洗成本、获客成本、销售周期、保修成本、盈亏平衡销量。

2027年：跑通可复制的业务闭环

销售：8~15台核心设备，8~12家真实付费客户，获得转介绍。建立标准化流程。

财务：年度毛利润为正，保留3~6个月运营资金。

产品：验证1~3种相关测试、维护或辅助小设备。新产品优先销售给现有客户群。

招聘：考虑兼职/远程助理，项目制技术支持，0~1名正式员工。

2028年：建立稳定产品组合和小型团队

销售规模达2027年1.5~2.5倍，形成2~3类产品，团队约3~5人，公司日常不再完全依赖创始人。

2029年前后：减少创始人日常执行

稳定客户和交付体系，创始人负责战略、关键客户、供应商关系和新项目。

四产品、客户与销售路径

1. 2026年目标客户

只集中验证以下四类企业：滚塑生产企业；塑料加工企业；橡胶加工企业；模具制造、维修和使用企业。

2026年不设其他行业探索组，不主动分散到船舶、焊接、普通除锈、文物修复等其他激光清洗场景。

2. 目标客户筛选标准

优先选择符合以下条件的企业：

  • 模具或设备需要反复清洗
  • 当前清洗耗时、费人工或需要停机
  • 模具价值较高，担心传统清洗损伤
  • 清洗频率较高
  • 激光清洗初步适用
  • 愿意安排生产、设备或管理人员沟通
  • 测试成功后存在采购可能
  • 位于可形成集中演示路线的区域
3. 客户漏斗与定义

截至2026年10月17日的建议目标：

  • 目标企业：30~50家
  • 已联系企业：25~40家（已通过邮件、电话、LinkedIn或行业关系触达）
  • 获得明确回复：10~20家
  • 有效客户访谈：10~15次（与了解生产、设备或模具清洗流程的人完成至少约20分钟沟通）
  • 合格客户机会：3~5家（有真实清洗痛点、激光初步适用、存在采购可能并愿意继续推进）
  • 愿意安排现场演示或生产试用：2~3家
4. CRM销售阶段管理

每个客户必须按照以下阶段推进和记录：

1目标企业列入名单
2已联系邮件、电话、LinkedIn触达
3有回复获得明确回复
4已访谈完成有效访谈
5合格机会通过筛选标准
6已演示完成现场演示
7生产试用3~5天试用
8已报价正式报价
9决策中等待采购决策
10成交 / 暂缓 / 失败

每个客户必须记录：当前阶段、负责人、下一步行动、下一步日期、预计采购时间、失败或暂停原因。

5. 失败原因分类

没有明显痛点 · 激光不适用 · 没有预算 · 采购周期太长 · 担心安全 · 价格过高 · 需要本地售后 · 已有替代方案 · 无法安排现场测试 · 时间不合适

6. 五阶段客户转化流程

阶段一：远程资格筛选

客户无需承担费用。通过电话、视频或邮件收集：工厂类型、模具和污染物照片、当前清洗方法、清洗频率、每次清洗人数和时间、是否影响生产、下一次计划清洗时间、生产/设备/维修/采购联系人。

只有以下客户才继续推进：有真实清洗问题；激光初步可能适用；当前方法存在明显成本或效率问题；愿意让真正了解生产的人参与；成功后存在采购可能。

阶段二：案例和视频沟通

使用中国工厂案例、清洗前后视频、时间和人工对比、安全与操作说明。目标不是马上成交，而是争取一次由生产、设备、维修或管理人员参与的技术沟通。

阶段三：免费现场小演示

对已筛选合格、位于重点区域或演示路线内的客户，安排约60~90分钟的小范围现场演示。

公司负责：携带演示设备、核心区域内正常送达、防护用品、初始参数、小区域清洗、清洗前后记录。

客户负责：提供真实模具/工件/代表性污染物、合适电源和安全作业区、安排了解生产和采购的人在场、提供原有清洗方式/时间和人工数据。

免费演示不对所有冷客户开放，需确认：有真实清洗任务、关键人员愿意到场、演示成功后愿意进一步试用、地理位置在合理路线内、存在实际采购可能。

阶段四：3~5天生产试用

小演示成功后，在客户真实清洗周期内将设备留在现场3~5个工作日。对外名称使用：Production Validation Trial / On-Site Production Trial / Try Before You Buy Program。不要将其主要描述为普通短租。

具体规则：送达当天检查现场安全；记录设备型号、序列号和状态；完成操作和安全培训；只允许指定人员使用；限定允许清洗的材料和工件；每天记录使用时间和清洗对象；提前定义成功标准；试用到期后按时回收；不允许无限期滞留；客户购买时可直接保留当前设备；不购买则按约定日期回收。

阶段五：试用转购买

试用结束当天安排复盘会议，讨论：是否达到清洗效果、是否节省时间和人工、是否减少停机、是否造成表面损伤、操作人员是否掌握、是否还有安全或技术顾虑、谁批准采购、预计决策日期。

客户选择：①直接购买试用机；②订购一台新机；③暂不购买，由公司按时回收。

7. 阶段性试用收费策略

前3~5个美国标杆客户：

  • 可以免费或极低费用提供生产试用
  • 公司承担核心区域内正常送达与回收
  • 目的不是赚租金，而是建立美国本地案例

客户必须提供非现金承诺：指定负责人；提供原清洗数据；决策人参与复盘；允许记录试用结果；成功后允许匿名或实名形成案例；在约定时间内作出购买或不购买决定；签署设备、安全和损坏责任协议。

拥有2~3个美国案例后：

  • 改为固定Production Trial Package
  • 包含送达、回收、培训和3~5天试用
  • 客户在15~30天内购买时，试用费部分或全部抵扣货款
  • 长途运输、住宿、特殊搬运和设备损坏不自动抵扣

2026年不开展长期租赁；短期试用是销售工具，不是独立租赁业务；不自行提供长期分期融资；客户需要融资时，再考虑第三方设备融资公司。

8. 美国标杆客户计划

2026年优先争取三类美国参考客户：

一家滚塑生产厂

可提供：免费或优惠生产试用、创始客户价格、更直接的技术支持、优先备件服务。

一家模具制造或维修企业

可提供：免费或优惠生产试用、创始客户价格、更直接的技术支持、优先备件服务。

一家其他塑料或橡胶加工企业

可提供：免费或优惠生产试用、创始客户价格、更直接的技术支持、优先备件服务。

希望换取：真实数据、清洗视频、客户评价、案例使用许可、推荐信、必要时允许潜在客户向其了解使用体验。

9. 区域演示策略

2026年不全国随机跑客户：

  1. 先锁定1~2个客户较集中的区域
  2. 每个区域筛选10~20家目标企业
  3. 提前4~6周安排演示路线
  4. 一次行程覆盖多家工厂
  5. 将小演示、生产试用和设备回收串联
  6. 原则上每次跨州行程至少安排2~4家客户交流

区域路线示例：

  • Midwest Mold Cleaning Demo Week
  • Ohio-Pennsylvania Production Trial Route
  • Rotomolding Customer Trial Week

不为一个完全陌生的冷客户单独横跨美国运输设备。同一区域客户数量达到一定程度后，再组织演示周。核心区域外客户等待下一次区域演示路线，或承担额外差旅成本。

10. 试用成功标准

每次现场演示或生产试用前，提前选择2~4项作为成功标准：

清洗效果
单位面积清洗时间
人工节省
停机减少
模具表面无损伤
操作学习时间
烟尘和现场影响
单次清洗成本

试用结束后必须根据上述指标复盘，不只用"效果不错"模糊评价。

同时确认客户是否允许：拍摄、使用匿名数据、使用客户名称、引用客户评价、让其他潜在客户联系了解体验。

11. 小型测试和辅助设备

2026年只记录需求，不增加产品。重点记录客户使用的检测设备、采购价格高或易损坏的产品、可一起交付的产品、被反复提及的需求。2027年再选择1~3种验证。

12. 品牌节奏

九售后与保修计划

12个月有限保修。三级支持体系：

  • 一级：远程诊断：视频检查、报警记录、参数日志、指导更换小配件
  • 二级：模块替换：镜片、激光头配件、控制器、电源模块、水冷配件、开关急停线缆
  • 三级：备用机或整机替换：重大故障且无法合理修复时

供应商应提供比客户保修期更长的支持，或提供明确免费备件额度。
