2026年是否已证明业务可行？

证明设备有效 → 做出可信案例 → 证明目标客户有问题 → 证明客户愿意试用或进一步了解 → 确认进口与安全可行 → 证明客户愿意付钱 → 证明毛利和售后可承担 → 扩大销售、产品和团队
2026年成功标准
最低标准：知道目标客户是谁、问题是否真实、设备是否有效、客户是否愿意继续试用或付钱，以及这项业务是否值得在2027年扩大。

理想成果：在完成上述验证的基础上，获得首批付费订单，并建立可以继续执行的公司、进口、销售和售后路径。

一创业总体定位

1. 长期公司定位

建立一家由本人独立拥有和管理、以中国供应链为基础、面向美国市场开展产品销售、贸易、市场进入及相关商业服务的公司。

公司长期不局限于：工业设备、激光清洗机、塑料加工行业、单一产品、单一销售模式。

未来可能开展：中国产品在美国的进口、代理和销售；针对美国企业的产品与解决方案；产品选型、试用、培训和售后；中国供应链采购与美国市场进入服务；品牌代理；其他通过市场验证的新产品与新项目。

2. 第一阶段业务切入口
第一阶段项目：以激光清洗设备为工具，为美国滚塑、塑料加工、橡胶加工及模具相关企业提供模具清洗、残留物去除和生产效率改善解决方案。

2026年优先客户：滚塑生产企业、其他塑料加工企业、橡胶加工企业、模具制造/维修和使用企业。暂不主动开发其他激光清洗行业。

3. 公司提供的不只是机器
  1. 了解客户当前如何清洗模具
  2. 判断污染物、材料和表面是否适合激光清洗
  3. 进行样品试洗或后续现场验证
  4. 比较传统方法和激光清洗的时间、人工、耗材及停机成本
  5. 推荐适合的设备型号和参数
  6. 销售设备和必要的安全配套
  7. 提供操作培训
  8. 提供有限保修、远程诊断和备件支持
  9. 根据同一批客户的真实需求，未来增加相关测试、维护或辅助设备

二2026年创业执行原则

1. 公司做宽，业务做窄

公司名称保持宽泛；2026年对外销售表达保持具体；不把公司永久限定为激光设备公司；也不向客户表达为"什么都卖的贸易公司"。

2. 必须完成的内容
  • 学会操作和基础调试机器
  • 完成真实工厂案例
  • 验证美国客户需求
  • 注册美国公司并建立收款能力
  • 个人账户和公司账户分离
  • 正式进口前确认产品合规和安全
  • 正式销售前明确合同、保修和售后责任
  • 算清售价、落地成本、毛利和现金流
  • 明确本人能够在美国开展哪些活动
3. 可以推迟的内容

2026年前期暂不处理：独立产品品牌、联邦商标注册、多个DBA、第二家LLC、豪华网站、办公室、长期仓库、大量库存、助理、全职员工、全职销售团队、其他行业探索、同时开发多条产品线。

4. 投入顺序
证明设备有效 → 做出可信案例 → 证明目标客户有问题 → 证明客户愿意试用或进一步了解 → 确认进口与安全可行 → 证明客户愿意付钱 → 证明毛利和售后可承担 → 扩大销售、产品和团队

三2026年至2029年长期目标

以下数字是第一版经营假设，必须在2026年10月17日完成美国市场验证后重新评估。

2026年：完成基础搭建与首轮验证

产品与客户：掌握设备操作，完成1个真实工厂案例，15~25次有效客户访谈，3~8个试洗/演示机会，1~3个付费订单。

公司：成立100%持有的美国公司，取得EIN，建立公司和个人银行账户，建立会计记录，明确进口销售路径，完成E-2可行性评估。

财务：不以盈利为标准，算清售价、单台落地成本、毛利、试洗成本、获客成本、销售周期、保修成本、盈亏平衡销量。

2027年：跑通可复制的业务闭环

销售：8~15台核心设备，8~12家真实付费客户，获得转介绍。建立标准化流程。

财务：年度毛利润为正，保留3~6个月运营资金。

产品：验证1~3种相关测试、维护或辅助小设备。新产品优先销售给现有客户群。

招聘：考虑兼职/远程助理，项目制技术支持，0~1名正式员工。

2028年：建立稳定产品组合和小型团队

销售规模达2027年1.5~2.5倍，形成2~3类产品，团队约3~5人，公司日常不再完全依赖创始人。

2029年前后：减少创始人日常执行

稳定客户和交付体系，创始人负责战略、关键客户、供应商关系和新项目。

五产品与客户发展计划

1. 2026年客户范围

只验证：滚塑生产企业、塑料加工企业、橡胶加工企业、模具制造/维修/使用企业。不设探索组，不主动分散到其他行业。选择标准：清洗痛点明显、频率较高、停机成本高、易理解激光清洗价值。

2. 小型测试和辅助设备

2026年只记录需求，不增加产品。重点记录客户使用的检测设备、采购价格高或易损坏的产品、可一起交付的产品、被反复提及的需求。2027年再选择1~3种验证。

3. 品牌节奏

2026年：一个宽泛公司名称 + 一个简单网站，不建独立品牌、不申请商标、不设置多个DBA。出现稳定订单后：再决定是否建立业务品牌、申请商标、拆分公司。

七招聘计划

2026年

不招聘助理，不招聘美国全职员工。本人主导全部工作。必要时购买短期项目服务：网站排版、视频剪辑、英文校对、临时技术支持。

2027年再考虑助理

条件：每周行政工作超过8~12小时并影响客户开发，CRM和行程有固定流程，公司可承担至少6个月成本。

技术人员

条件：有2~3个真实演示，获得第一笔订单，客户要求美国本地技术支持。

第一名全职员工

条件：已有真实付费客户，有明确未来工作量，招聘后仍有6个月以上运营资金，岗位可明显提高销售、交付或客户体验。

八资金计划

第一阶段 10万~20万元

7月~10月：江西学习、工厂案例、网站和销售资料、公司基础准备、美国差旅和生活、手机与银行、客户拜访、基础样品和材料

第二阶段 25万~45万元

启动条件：产品案例可信 + 3~5个真实客户机会 + 客户愿意进一步试洗或演示 + 初步确认进口和安全可行。用途：演示设备、安全附件、烟尘处理、运输和报关、专业顾问、产品责任保险、备件、车辆或运输方案

第三阶段 30万~60万元

启动条件：获得订单 / 确定正式E-2方案 / 市场反馈明显支持扩大。用途：运营储备、备用设备、E-2申请、展会、备件库存、2027年新产品验证、后续招聘

第一年暂定资金授权上限：65万~125万元人民币，分阶段释放。

九售后与保修计划

12个月有限保修。三级支持体系：

  • 一级：远程诊断：视频检查、报警记录、参数日志、指导更换小配件
  • 二级：模块替换：镜片、激光头配件、控制器、电源模块、水冷配件、开关急停线缆
  • 三级：备用机或整机替换：重大故障且无法合理修复时

供应商应提供比客户保修期更长的支持，或提供明确免费备件额度。
